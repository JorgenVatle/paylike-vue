# Paylike Vue
A Vue wrapper for the [Paylike SDK](https://github.com/paylike/sdk).

## Installation
```bash
npm install --save paylike-vue
```

## Import
```js
import PaylikeVue from 'paylike-vue';

Vue.use(PaylikeVue);
```
